请求参数
字段名	含义	备注	必填
appKey	应用标识（应用 ID）	**** 325f2a1d4****	true
curtime	时间戳	当前UTC时间戳(秒)	true
q	正文文本	图片文本	true
salt	随机字符串	推荐使用uuid确保唯一性	true
sign	签名信息：sha256(appKey+input+salt+密钥)	sha256(appKey+input+salt+密钥)	true
signType	签名类型	v3	true
grade	作文等级	取值见grade	false
title	作文标题和题目要求	标题内容+题目要求，5000字符以内	false
isNeedSynonyms	是否查询同义词	默认为 false（true/false）	false
correctVersion	作文批改版本：基础，高级	默认为 basic（basic/advanced）返回结果见示例	false
isNeedEssayReport	是否返回写作报告(essayReport字段,仅在高级批改时生效)	默认为 false（true/false）	false
最好传输 limitedWords，这样评分更精确。 签名生成算法如下：

signType=v3，sha256(应用 ID+input+salt+curtime+密钥)，推荐使用 sha256 签名计算方法为：sha256(应用 ID+input+salt+当前 UTC 时间戳+密钥)。

其中，input 的计算方式为：input=多个q拼接后前10个字符 + 多个q拼接长度 + 多个q拼接后十个字符（当多个 q 拼接后长度大于 20）或 input=多个q拼接的字符串（当多个 q 拼接后长度小于等于 20）。
