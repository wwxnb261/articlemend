使用说明

由于API调用需要密钥认证，请按照以下步骤操作：

  1. 打开app.js文件
  2. 找到顶部的配置部分，填写您的有道智云应用ID和密钥： 
    // 配置信息
const APP_KEY = '您的应用ID';
const APP_SECRET = '您的应用密钥';
  3. 保存文件后刷新页面即可使用

注意：请确保您的应用已开通英语作文批改API权限

关于跨域问题的说明：

此应用可能会遇到跨域问题。我们提供了一个本地代理服务器解决方案：

  1. 安装Node.js（如果尚未安装）
  2. 打开命令行，进入项目目录
  3. 运行命令：npm install 安装依赖
  4. 运行命令：npm start 启动代理服务器
  5. 点击下方的"切换到代理模式"按钮
  6. 刷新页面即可使用

代理服务器会在本地运行，无需访问国外网站，可以有效解决跨域问题。
